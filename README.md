# SmartHostel — Intelligent Hostel Room Allocation System

## Project Overview

SmartHostel is a Java-based Hostel Room Allocation System developed using Object-Oriented Programming principles.

The system manages:

* Student registrations
* Hostel room allocation
* Room availability
* Complaint management
* Occupancy tracking

The project provides an efficient solution for handling hostel operations through an interactive console-based application.

---

# Features

* Student Registration System
* Room Management
* Automatic Room Allocation
* Complaint Registration
* Room Availability Tracking
* Occupancy Management
* Interactive Console Interface

---

# OOP Concepts Used

| Concept        | Implementation           |
| -------------- | ------------------------ |
| Class & Object | Student, Room, Complaint |
| Inheritance    | Student extends User     |
| Encapsulation  | Private data members     |
| Polymorphism   | Method handling          |
| Abstraction    | Hostel operations        |

---
